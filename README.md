📱 Smartphone Sales Analysis Dashboard

📌 Project Overview

This project analyzes a smartphone dataset using Python, SQL, and Tableau to extract insights
about pricing, ratings, discounts, and brand performance.

The goal is to transform raw data into meaningful insights and build an interactive dashboard.

🛠️ Tools & Technologies Used

Python (Pandas, Matplotlib, Seaborn) → Data Cleaning & EDA
SQL Server → Data Storage & Querying
Tableau → Data Visualization & Dashboard
GitHub → Project Documentation


📂 Dataset
File: SmartPhones_Sales.csv
Contains:
Brand
Model
Price
Discount
Rating


🧹 Data Cleaning (Python)

Steps performed:

Removed duplicates
Handled missing values
Converted price & discount to numeric
Created new columns:
final_price
value_score

📊 Exploratory Data Analysis (Python)

Key analysis:

Price distribution
Price vs Rating
Top brands by rating
Best value phones

🗄️ SQL Analysis

Database: SmartphoneDB

Key Queries:

Top brands by average rating
Brand-wise average price
Top 10 best value phones
Discount analysis
Price category segmentation

📈 Tableau Dashboard

🔹 Key Features:

KPI Cards (Avg Price, Rating, Total Phones, Discount)
Price vs Rating Scatter Plot
Brand Performance Chart
Price Category Distribution
Best Value Phones
Discount Analysis

🎯 Calculated Fields (Tableau)

Price Category:
Budget / Mid Range / Premium
Value Category:
High / Medium / Low
Rating Category:
Excellent / Good / Average

🧠 Key Insights

Premium phones have higher prices but not always higher ratings
Mid-range phones provide the best value
Some brands offer higher discounts than others
Ratings are mostly concentrated between 4.0 – 4.5
